حول المشروع

هذا المشروع يستخدم نموذج CATT (Context-Aware Text Tashkeel) المبني على تقنيات التعلم العميق لإضافة الحركات (التشكيل) على النصوص العربية بشكل آلي وبدقة عالية.

المميزات

  • ✅ تشكيل آلي دقيق باستخدام الذكاء الاصطناعي
  • ✅ وضعان: سريع (EncoderOnly) ودقيق (EncoderDecoder)
  • ✅ دعم معالجة الملفات النصية
  • ✅ إحصائيات تفصيلية عن التشكيل
  • ✅ واجهات متعددة: Streamlit، Gradio، صفحة ويب
  • ✅ إمكانية إزالة التشكيل

التشغيل

# Streamlit

streamlit run app_streamlit.py


# Gradio

python app_gradio.py
